Customer Segmentation Project 

Customer Segmentation using Python and Power BI


 Project Overview

This project focuses on customer segmentation using Machine Learning and data visualization techniques. Customers are grouped based on their income and spending behavior using the K-Means Clustering algorithm.

The project helps businesses understand different customer groups and make better marketing decisions.


Technologies Used

 Python Libraries

* Pandas
* Matplotlib
* Scikit-learn

 Tools

* Jupyter Notebook
* Power BI



Features

* Customer data analysis
* Data cleaning and preprocessing
* K-Means clustering
* Customer segmentation visualization
* Interactive Power BI dashboard
* Cluster-based business insights

Python Implementation

 Steps Performed

1. Imported customer dataset
2. Cleaned income data
3. Applied K-Means clustering
4. Generated customer clusters
5. Visualized clusters using scatter plot






 K-Means Clustering

K-Means algorithm groups similar customers together based on:

 Income
 Spending Score
 
The algorithm creates clusters automatically and identifies customer behavior patterns.
